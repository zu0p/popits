# POPITS
## 실행 방법
1. backend 폴더에서 
  ```node app.js```
  로 서버 실행
2. frondend 폴더에서 
  ```npm start```
  로 프론트 실행
3. ```/profile``` 에서 스티팝 API 활용한 프로필이미지 변경 확인 가능
